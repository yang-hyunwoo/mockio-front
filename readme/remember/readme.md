## 🔐 비밀번호 찾기 / 재설정

[🔝 메인 목차로 이동](../../readme.md)

가입한 이메일을 통해 비밀번호 재설정 링크를 발송하고, 사용자는 해당 링크에서 새로운 비밀번호를 설정할 수 있습니다.  
보안을 위해 재설정 요청 결과는 계정 존재 여부와 관계없이 유사한 안내 문구로 처리하며, 링크는 1회용이며 일정 시간이 지나면 만료됩니다.

---
### 1. 비밀번호 재설정 이메일 요청

사용자가 가입한 이메일 주소를 입력하면 비밀번호 재설정 링크를 전송합니다.  
일반 회원가입 계정만 재설정이 가능하며, 소셜 로그인 계정은 해당 로그인 제공자를 통해 로그인해야 합니다.

<img src="../image/비밀번호변경이메일입력.png" alt="비밀번호 재설정 이메일 입력 화면">

#### 주요 특징
- 가입한 이메일 주소로 재설정 링크 전송
- 계정 존재 여부와 관계없이 유사한 응답 제공
- 소셜 로그인 계정은 비밀번호 재설정 비지원 안내 표시
- 재설정 링크는 1회용으로 사용 가능
---

### 2. 이메일 발송 요청 완료

이메일 전송 요청이 완료되면 사용자에게 메일함 확인을 안내합니다.  
메일이 보이지 않을 경우 스팸함도 함께 확인할 수 있도록 안내합니다.

<img src="../image/비밀번호변경요청완료.png" alt="비밀번호 재설정 요청 완료 화면">


---

### 3. 재설정 메일 수신

사용자는 이메일로 전달된 비밀번호 재설정 링크를 통해 비밀번호 변경 페이지로 이동할 수 있습니다.  
링크에는 만료 시간이 적용되며, 요청하지 않은 경우 메일을 무시할 수 있도록 안내합니다.

<img src="../image/비밀번호이메일.png" alt="비밀번호 재설정 이메일">

---

### 4. 새 비밀번호 입력

사용자는 새 비밀번호와 비밀번호 확인 값을 입력하여 비밀번호를 재설정할 수 있습니다.  
비밀번호는 영문, 숫자, 특수문자를 포함한 8자 이상 규칙을 따릅니다.

<img src="../image/비밀번호변경.png" alt="새 비밀번호 입력 화면">

#### 검증 항목
- 새 비밀번호 입력
- 새 비밀번호 확인 입력
- 비밀번호 규칙 충족 여부 확인
- 비밀번호 확인 일치 여부 확인

---

### 5. 비밀번호 재설정 완료

비밀번호 변경이 완료되면 성공 메시지를 표시하고, 사용자가 새로운 비밀번호로 다시 로그인할 수 있도록 로그인 화면으로 이동을 유도합니다.

<img src="../image/비밀번호변경완료.png" alt="비밀번호 재설정 완료 화면">

---

### ✅ 보안 처리 사항
- 비밀번호 재설정 링크는 1회용으로 사용
- 일정 시간이 지나면 링크 만료
- 계정 존재 여부 노출 방지
- 소셜 로그인 계정은 별도 재설정 불가
- 비밀번호 변경 완료 후 재로그인 유도
